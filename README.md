# LaunchPilot RN 🚀

Premium React Native SaaS Starter Kit with Expo Router, Zustand, TailwindCSS (NativeWind), TypeScript, and Supabase.

Designed to help developers launch and scale mobile SaaS applications in record time.

## Tech Stack
- **Framework:** Expo SDK, Expo Router, TypeScript
- **Styling:** NativeWind, TailwindCSS
- **State:** Zustand
- **Database / Auth / Storage:** Supabase
- **Forms & Validation:** React Hook Form + Zod
- **Animations:** Moti, Reanimated
- **Payments:** Razorpay + Stripe scaffolds

## Getting Started
1. Install dependencies:
   ```bash
   npm install --legacy-peer-deps
   ```
2. Set up environment:
   Copy `.env.example` to `.env` and fill in credentials.
3. Start development server:
   ```bash
   npx expo start
   ```

Refer to `/docs` directory for detailed documentation on integrations, deployment, and customization.
